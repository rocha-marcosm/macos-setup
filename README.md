# macos-setup
Shell Script to set up a new MacBook


## Setup

```bash
/bin/zsh -c "$(curl -fsSL https://raw.githubusercontent.com/rocha-marcosm/macos-setup/v0.1/config.sh)"
```

## Packages upgrade

```bash
/bin/zsh -c "$(curl -fsSL https://raw.githubusercontent.com/rocha-marcosm/macos-setup/v0.1/brew.sh)"
```
